# 019 - 픽셀 아트 스튜디오

## 컨셉
브라우저에서 즐기는 픽셀 아트 에디터 겸 창작 도구. 16x16부터 64x64까지의 캔버스에서 픽셀 아트를 그리고, 프레임 애니메이션을 만들고, PNG/GIF로 내보내기할 수 있다. 게임이라기보다 창작 도구이지만, "오늘의 주제" 챌린지와 갤러리 기능으로 게임적 요소를 가미한다.

## 게임 규칙
- 캔버스 크기 선택: 16x16, 32x32, 64x64
- 도구: 연필, 지우개, 채우기(flood fill), 라인, 사각형, 원, 스포이드, 선택/이동
- 레이어 시스템: 최대 4개 레이어, 순서 변경/투명도 조절 가능
- 팔레트: 기본 16색 팔레트 제공, 커스텀 색상 추가 가능
- 애니메이션: 최대 12프레임, 어니언 스킨(이전 프레임 반투명 표시) 지원
- 내보내기: PNG(단일 프레임), 스프라이트 시트(전체 프레임 가로 연결), GIF 애니메이션
- 대칭 모드: 좌우/상하/4방향 대칭 그리기
- 실행 취소/다시 실행: 최대 50단계

## 조작법
- PC: 마우스 왼클릭 그리기, 오른클릭 스포이드, 마우스 휠 확대/축소, 키보드 단축키(B:펜, E:지우개, G:채우기, Z:취소, Shift+Z:다시실행)
- 모바일: 터치 그리기, 두 손가락 핀치 줌, 하단 도구바에서 도구 선택

## 비주얼 스타일
- UI: 다크 테마 (#1e1e2e 배경, #313244 패널)
- 캔버스 배경: 체커보드 패턴 (투명 영역 표시)
- 격자선: 1px 반투명 회색
- 도구 아이콘: CSS로 만든 간단한 심볼
- 선택된 도구/색상은 밝은 보라 (#cba6f7) 하이라이트
- 전체적으로 VS Code 느낌의 모던 에디터 UI

## 핵심 시스템

### 캔버스 데이터 구조
```javascript
// 각 레이어는 2D 배열, 각 셀은 RGBA 색상 또는 null(투명)
const canvas = {
  width: 32,
  height: 32,
  layers: [
    { name: 'Layer 1', visible: true, opacity: 1.0, data: Array(32*32).fill(null) },
    // ...
  ],
  activeLayer: 0,
  frames: [ /* 각 프레임은 layers 배열의 deep copy */ ],
  activeFrame: 0
};

// 색상 형식: { r, g, b, a } (0-255)
```

### Flood Fill 알고리즘
```
BFS 기반:
1. 시작 픽셀의 색상을 targetColor로 저장
2. 큐에 시작 좌표 추가
3. 큐에서 좌표를 꺼내 상하좌우 인접 픽셀 확인
4. 인접 픽셀 색상이 targetColor와 같으면 새 색상으로 변경 후 큐에 추가
5. 큐가 빌 때까지 반복
- 같은 좌표 중복 방문 방지를 위해 visited Set 사용
```

### 실행 취소 시스템
```
- undoStack: 변경 전 상태를 저장하는 스택 (최대 50)
- redoStack: 취소된 상태를 저장하는 스택
- 변경 단위: 마우스 다운~업 사이의 모든 픽셀 변경을 하나의 액션으로 묶음
- 각 액션: { layerIndex, changes: [{x, y, oldColor, newColor}, ...] }
- 취소 시 changes를 역순으로 oldColor 적용, 다시실행 시 newColor 적용
```

### PNG 내보내기
```
1. 임시 캔버스 생성 (원본 크기)
2. 레이어를 아래→위 순서로 합성 (globalAlpha로 투명도 적용)
3. canvas.toDataURL('image/png')으로 data URL 생성
4. <a> 태그의 download 속성으로 자동 다운로드 트리거
```

### GIF 내보내기 (간소화)
```
- 순수 JS GIF 인코더 구현 (LZW 압축)
- 또는 각 프레임을 PNG로 내보내는 스프라이트 시트 방식 사용
- 단순화를 위해 스프라이트 시트 PNG 내보내기를 기본으로 하고,
  브라우저에서 프레임 재생 미리보기 제공
```

## 기술 구현
- 그리기 캔버스: Canvas 2D API, 줌 레벨에 따라 픽셀을 확대 렌더링
- 격자: strokeRect로 각 픽셀 경계 그리기 (줌 4x 이상일 때만 표시)
- 투명 배경: 별도 캔버스에 체커보드 패턴을 createPattern으로 생성
- 도구 동작: pointerdown/pointermove/pointerup 이벤트로 통합 처리
- 대칭 그리기: 그리기 좌표를 변환하여 대칭 위치에도 동시 적용
- 레이어 합성: 각 레이어 캔버스를 globalAlpha와 함께 메인 캔버스에 drawImage
- 미리보기: 별도 작은 캔버스에 1x 스케일로 합성 결과 표시
- 애니메이션 재생: setInterval로 프레임 순환, 어니언 스킨은 이전 프레임을 30% 투명도로 오버레이
- 반응형: CSS Grid로 패널 배치, 모바일에서는 사이드 패널을 하단으로 접기
