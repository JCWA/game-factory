# 001 - 스네이크 게임

## 컨셉
클래식 스네이크 게임의 현대적 리메이크. 네온 스타일 비주얼.

## 게임 규칙
- 뱀이 자동으로 전진, 방향키로 방향 전환
- 먹이(사과)를 먹으면 몸이 1칸 성장 + 점수 획득
- 벽 또는 자기 몸에 부딪히면 게임오버
- 먹이 5개마다 속도 증가 (난이도 상승)

## 조작법
- **PC**: 방향키 (←↑↓→) 또는 WASD
- **모바일**: 스와이프 (상하좌우)

## UI 구성

```
┌─────────────────────────┐
│  SCORE: 0    BEST: 0    │
├─────────────────────────┤
│                         │
│         게임 보드        │
│       (20x20 그리드)     │
│                         │
│                         │
├─────────────────────────┤
│   [시작] [다시하기]      │
└─────────────────────────┘
```

## 비주얼 스타일
- 다크 배경 (#1a1a2e)
- 네온 그린 뱀 (#00ff41)
- 빨간 먹이 (#ff0040)
- 그리드 라인 은은하게 표시
- 게임오버 시 화면 페이드 + 점수 표시

## 기술 구현
- Canvas API로 렌더링
- requestAnimationFrame 기반 게임 루프
- localStorage로 최고 점수 저장
- 단일 index.html 파일
