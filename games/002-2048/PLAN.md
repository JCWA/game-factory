# 002 - 2048

## 컨셉
4x4 그리드에서 숫자 타일을 밀어 합치는 퍼즐 게임. 2048 타일을 만들면 승리.

## 게임 규칙
- 4x4 그리드에 랜덤으로 2 또는 4 타일 생성
- 상하좌우로 밀면 모든 타일이 해당 방향으로 이동
- 같은 숫자 타일이 만나면 합쳐짐 (2+2=4, 4+4=8, ...)
- 한 번의 이동에서 한 타일은 한 번만 합쳐짐
- 이동 후 빈 칸에 새 타일(2 또는 4) 생성
- 2048 타일 생성 시 승리 (계속 플레이 가능)
- 이동 불가 시 게임오버

## 조작법
- **PC**: 방향키 (←↑↓→) 또는 WASD
- **모바일**: 스와이프 (상하좌우)

## 비주얼 스타일
- 따뜻한 톤 배경 (#faf8ef)
- 타일 색상: 숫자별 고유 색상 (2=연한 베이지 ~ 2048=골드)
- 타일 합칠 때 팝 애니메이션
- 새 타일 등장 시 페이드인
- 부드러운 라운드 코너
