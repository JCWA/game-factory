# 027 - 우주 탐사

## 컨셉
태양계 시뮬레이션 기반 우주 탐사 게임. 실제 중력 물리를 적용한 궤도 역학으로 우주선을 발사하고, 행성을 탐사하며, 자원을 수집하고, 기지를 건설한다. Kerbal Space Program의 핵심 재미를 2D로 축약. "궤도를 설계하는" 전략적 재미와 "미지의 행성을 탐사하는" 탐험의 재미를 결합.

핵심 재미: 중력을 이용한 궤도 기동의 물리적 쾌감 + 절차적으로 생성된 행성 탐사의 탐험 재미 + 자원 관리의 긴장감.

## 핵심 시스템

### 시스템 1: 궤도 역학 & 중력 시뮬레이션
- **2D 중력 시뮬레이션**: 뉴턴 만유인력 법칙
  ```
  F = G * M1 * M2 / r²
  가속도 a = G * M / r²  (우주선 질량 무시)
  ```
- **적분**: Verlet 적분법 (오일러보다 안정적)
  ```
  position(t+dt) = 2*position(t) - position(t-dt) + acceleration(t)*dt²
  ```
- **시간 스케일**: 실시간 x1 ~ x1000 (와프)
- **태양계 구성** (실제 비율 축소):

| 천체 | 질량(상대) | 궤도반경(px) | 공전주기 | 색상 | 크기(px) |
|------|-----------|-------------|---------|------|---------|
| 태양 | 10000 | 0 (중심) | - | #FFD700 | 30 |
| 수성 | 5 | 80 | 8초 | #A0522D | 4 |
| 금성 | 15 | 130 | 15초 | #DEB887 | 6 |
| 지구 | 15 | 190 | 25초 | #4169E1 | 7 |
| 화성 | 8 | 260 | 40초 | #CD5C5C | 5 |
| 목성 | 200 | 380 | 80초 | #DAA520 | 15 |
| 토성 | 150 | 500 | 120초 | #F4A460 | 13 |

- **달**: 지구 주위를 도는 달 1개 (질량 2, 궤도반경 20px)

- **궤도 예측선**: 현재 속도로 100스텝 미래의 경로를 점선으로 표시. 추력 방향 변경 시 실시간 업데이트.

- **호만 전이 궤도 가이드**: 두 행성 간 이동 시 최적 발사 시점과 필요 델타-V를 계산하여 표시.

### 시스템 2: 우주선 관리
- **우주선 속성**:
  ```
  ship = {
    x, y, vx, vy,          // 위치와 속도
    fuel: 100,              // 연료 (0~100)
    hull: 100,              // 선체 내구도
    cargo: [],              // 수집한 자원
    cargoCapacity: 50,      // 화물 용량
    thrustPower: 0.5,       // 추력
    modules: []             // 장착 모듈
  }
  ```

- **추력 시스템**: 
  - 방향키/WASD로 추력 방향 제어
  - 추력 크기 조절 (10%, 50%, 100%)
  - 연료 소비 = thrustPower * dt * throttle
  - 연료 0이면 관성 비행만 가능

- **모듈 시스템** (5종):
  | 모듈 | 효과 | 획득 |
  |------|------|------|
  | 대형 연료탱크 | 연료 용량 +50 | 화성 기지 건설 |
  | 고효율 엔진 | 연료 소비 -30% | 목성 탐사 |
  | 화물 확장 | 화물 용량 +30 | 금성 탐사 |
  | 방어막 | 소행성 충돌 무효화 | 토성 탐사 |
  | 스캐너 | 행성 자원 사전 탐지 | 수성 탐사 |

### 시스템 3: 행성 탐사 (절차적 생성)
각 행성에 도착하면 2D 사이드뷰 탐사 모드로 전환:

- **행성 표면 생성**: 
  - Perlin noise로 지형 높이맵 생성 (폭 800px)
  - 행성별 특성:
    - 수성: 크레이터 많음, 자원 풍부, 높은 온도
    - 금성: 평탄, 산성 안개 (시야 제한), 독특한 광물
    - 화성: 사막, 계곡, 물 흔적, 건설 최적지
    - 목성: 착륙 불가 (가스 행성), 위성만 탐사
    - 토성: 착륙 불가, 고리에서 자원 채취

- **탐사 조작**: 좌우 이동 + 점프 (중력은 행성별 다름)
- **자원 채취**: 지형 아래 자원 노드를 발견하면 채취 (시간 소요)
- **자원 종류 6개**:
  | 자원 | 주 출현 행성 | 용도 |
  |------|-------------|------|
  | 철 | 수성, 화성 | 기지 건설 |
  | 헬륨-3 | 달, 목성 | 연료 제조 |
  | 물 | 화성, 토성 | 생존 / 연료 |
  | 금 | 수성 | 업그레이드 |
  | 티타늄 | 금성 | 고급 건설 |
  | 크리스탈 | 토성 | 특수 모듈 |

### 시스템 4: 기지 건설
- 화성에 메인 기지 건설 가능 (조건: 철 20 + 물 10)
- **기지 시설**:
  | 시설 | 비용 | 기능 |
  |------|------|------|
  | 거주 모듈 | 철 10 | 기지 활성화 (필수) |
  | 연료 공장 | 철 15, 물 5 | 물 → 연료 변환 (5:20) |
  | 연구실 | 철 20, 금 5 | 모듈 연구 해금 |
  | 발사대 | 철 30, 티타늄 10 | 화성에서 재발사 가능 |
  | 통신탑 | 철 10, 크리스탈 3 | 행성 정보 원격 확인 |

### 시스템 5: 이벤트 & 위험 시스템
- **소행성대**: 화성-목성 사이. 소행성이 랜덤 궤도로 이동. 충돌 시 선체 -20.
- **태양풍**: 랜덤 이벤트. 태양 방향으로 밀어냄. 30초 지속.
- **연료 부족 경고**: 연료 < 20% 시 경고. 가장 가까운 연료 보급 지점 표시.
- **행성 착륙 실패**: 속도 > 임계값으로 접근 시 충돌 (선체 데미지). 적정 진입 속도를 게이지로 표시.

## 게임 규칙
- 지구 궤도에서 시작. 초기 연료 100, 화물 0.
- 행성 궤도에 진입 (거리 < 행성크기 * 3) 하면 "탐사" 버튼 활성화
- 탐사 시 우주선은 행성 궤도에 정박
- 행성 표면에서 자원을 채취하여 우주선에 적재
- 화물이 가득 차면 더 이상 채취 불가
- 지구로 귀환하여 자원을 기지 건설에 사용
- 화성 기지 완성이 최종 목표 (모든 시설 건설)
- 선체 0이면 게임 오버 (파편이 되어 떠다님)
- 연료 0이면 관성으로만 이동 (구조 불가, 사실상 게임 오버)
- 시간 와프 중 행성 근접 시 자동으로 와프 해제
- 미니맵에서 전체 태양계 확인 가능
- 각 행성 첫 탐사 시 발견 로그 기록

## AI 설계

### 궤도 어시스턴트 AI
플레이어에게 궤도 기동을 도와주는 가이드 시스템:

```
1. 호만 전이 계산기:
   - 현재 궤도와 목표 행성 궤도의 반장축 계산
   - 필요 델타-V = sqrt(GM/r1) * (sqrt(2*r2/(r1+r2)) - 1)  (1차 분사)
   - 도착 시 원형궤도 진입 델타-V 계산
   - 최적 발사 타이밍 = 행성 위상각 기반

2. 충돌 경고 시스템:
   - 매 프레임 100스텝 미래 예측
   - 예측 경로가 천체/소행성과 교차하면 경고
   - 회피 방향 화살표 표시

3. 착륙 가이드:
   - 행성 접근 시 상대 속도 표시
   - 안전 속도 범위를 게이지로 표시 (초록/노랑/빨강)
   - 역추진 타이밍 안내
```

### 소행성 AI
- 각 소행성은 독립적 궤도 운동 (타원 궤도)
- 소행성 20~30개가 소행성대에 분포
- 궤도 매개변수 (반장축, 이심률, 초기 위상) 랜덤 생성
- 충돌 판정: 원-원 교차 검사

## 비주얼 스타일
- **배경**: #0a0a2e (깊은 우주). 별 100개 랜덤 배치 (1px 흰 점, 깜빡임).
- **태양**: 방사형 그라디언트 (#FFD700 → #FF8C00), 글로우 효과.
- **행성**: 각 행성 고유 색상 + 반투명 대기 효과 (arc 이중 그리기).
- **토성 고리**: 타원 arc로 표현.
- **궤도선**: 반투명 흰색 점선.
- **우주선**: 삼각형 (▲). 추력 시 뒤에 주황 불꽃 파티클.
- **소행성**: 불규칙 다각형 (회색).
- **행성 탐사**: 각 행성 색조의 그라디언트 하늘 + 지형.
- **UI**: 반투명 검정 (rgba 0,0,0,0.7). 네온 블루 (#00d4ff) 강조.

## 레벨/콘텐츠 설계

### 진행 흐름
1. **지구 출발** → 달 탐사 (튜토리얼: 궤도 기동 학습)
2. **내행성 탐사** → 수성, 금성 (자원 수집, 모듈 획득)
3. **화성 기지 건설** → 화성 탐사 + 기지 시설 건설
4. **외행성 탐사** → 목성, 토성 (위험하지만 보상 큼)
5. **완성** → 모든 기지 시설 건설 완료

### 발견 로그 콘텐츠 (행성별 3-5개)
- 수성: "태양에 가장 가까운 세계. 극심한 온도 차이가 광물을 결정화시켰다."
- 금성: "두꺼운 대기 아래 숨겨진 보물. 산성 구름 사이로 티타늄 광맥이 보인다."
- 화성: "붉은 사막 아래 얼어붙은 물의 흔적. 이곳이 우리의 새 터전이 될 것이다."
- 목성: "거대한 가스 행성의 위성에서 헬륨-3를 발견. 무한 에너지의 열쇠."
- 토성: "아름다운 고리는 얼음과 크리스탈의 보물창고였다."

## 기술 구현 계획

### 도전 과제 및 해결
1. **궤도 역학 안정성**: Verlet 적분으로 에너지 보존 개선. dt를 작게 (0.016) 유지. 와프 시 dt는 고정, 루프 횟수를 늘림.
2. **궤도 예측선**: 현재 상태를 복사하여 100스텝 시뮬레이션. 매 프레임 재계산은 무겁므로 추력 변경 시에만 업데이트.
3. **줌 레벨**: 태양계 전체 (zoom=0.2) ~ 행성 근접 (zoom=5). 줌 레벨에 따라 렌더링 상세도 조절.
4. **모드 전환**: solar ↔ planet ↔ base 모드 전환 시 캔버스 리드로우. 각 모드별 별도 렌더링/입력 처리 함수.
5. **Perlin Noise (행성 지형)**: 1D perlin noise로 표면 높이맵 생성. seed를 행성별로 다르게.
