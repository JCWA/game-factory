# 022 - 우주 표류기 (비주얼 노벨)

## 컨셉
고장 난 우주선에서 살아남아야 하는 SF 인터랙티브 픽션. 플레이어는 우주선의 유일한 생존자로, 제한된 자원(산소, 식량, 전력)을 관리하며 선택지를 골라 스토리를 진행한다. 7개의 엔딩이 있으며, 선택에 따라 생존/구조/비극 등 다양한 결말로 분기한다. 텍스트 + 배경 일러스트 + 상태 관리 시스템이 핵심.

## 게임 규칙
- 10일(10챕터) 동안 생존해야 한다
- 매 챕터마다 2-3개의 선택지가 주어진다
- 자원 3종: 산소(100), 식량(100), 전력(100) - 매일 일정량 소모
- 선택에 따라 자원이 증감하고 스토리가 분기한다
- 자원 중 하나라도 0이 되면 해당 자원 관련 배드 엔딩
- 특정 선택 조합으로만 도달하는 히든 엔딩이 존재
- NPC와의 통신(AI 봇, 구조대, 미스터리 신호)이 스토리에 영향
- 이전 선택이 이후 선택지와 이벤트에 영향을 미침 (플래그 시스템)
- 엔딩 도달 후 "엔딩 도감"에 기록되어 수집 요소 제공

## 조작법
- PC: 마우스 클릭으로 선택지 고르기, 스페이스바/엔터로 텍스트 넘기기
- 모바일: 터치로 선택지 고르기, 탭으로 텍스트 넘기기

## 비주얼 스타일
- 배경: CSS 그라디언트와 기하학적 도형으로 구성한 미니멀 SF 일러스트
  - 우주선 내부: 어두운 금속 톤 (#1a1a2e → #16213e), 비상등 붉은 점멸
  - 우주 외부: 깊은 검정 + 별 파티클 + 행성/성운 그라디언트
  - 통신실: 녹색 모니터 글로우
- 텍스트 박스: 반투명 검정 (#000000cc), 밝은 텍스트
- 선택지: 호버 시 시안 (#00d4ff) 하이라이트, 선택 시 펄스 애니메이션
- 자원 바: 산소=시안, 식량=녹색, 전력=노란색
- 타이핑 효과: 텍스트가 한 글자씩 나타남 (30ms 간격)
- 장면 전환: 1초 페이드 인/아웃

### 자원 관리 시스템
```javascript
const resources = {
  oxygen: 100,   // 매일 -10, 재순환장치 작동 시 -5
  food: 100,     // 매일 -10, 식물실 가동 시 -5
  power: 100     // 매일 -8, 태양전지 수리 시 매일 +5
};

// 매 챕터 종료 시 일일 소비 적용
function dailyConsumption(state) {
  state.oxygen -= state.flags.recyclerFixed ? 5 : 10;
  state.food -= state.flags.gardenActive ? 5 : 10;
  state.power -= 8;
  if (state.flags.solarFixed) state.power += 5;
  
  // 자원 0 이하 체크 → 배드 엔딩 분기
  if (state.oxygen <= 0) return 'ending_suffocation';
  if (state.food <= 0) return 'ending_starvation';
  if (state.power <= 0) return 'ending_darkness';
  return null;
}
```

### 엔딩 시스템 (7개)
```
1. 질식 엔딩 - 산소 고갈
2. 굶주림 엔딩 - 식량 고갈
3. 암흑 엔딩 - 전력 고갈
4. 구조 엔딩 - 구조 신호 발신 성공, 10일째 구조대 도착
5. 자력 귀환 엔딩 - 엔진 수리 성공, 직접 귀환
6. 미스터리 엔딩 - 미지의 신호를 따라간 결과
7. 완벽 엔딩 (히든) - 모든 시스템 수리 + NPC 전원 구출 + 미스터리 해결
```

### 플래그 시스템
```javascript
const flags = {
  askedAria: false,       // 아리아에게 상황 물어봄
  recyclerFixed: false,   // 산소 재순환장치 수리
  gardenActive: false,    // 식물 재배실 가동
  solarFixed: false,      // 태양전지 수리
  distressSignal: false,  // 구조 신호 발신
  engineRepaired: false,  // 엔진 수리
  mysterySignal: false,   // 미스터리 신호 수신
  crewFound: 0,           // 발견한 동료 수 (0-3)
  trustAria: 0            // 아리아 신뢰도 (-3 ~ +3)
};
```

## 기술 구현
- DOM 기반 렌더링 (텍스트와 선택지는 HTML 요소)
- 배경: div 레이어에 CSS 그라디언트 + pseudo-element로 도형 배치
- 별 파티클: 작은 div를 absolute 포지션으로 배치, CSS animation으로 반짝임
- 타이핑 효과: setInterval로 텍스트를 한 글자씩 innerHTML에 추가
- 상태 관리: 단일 gameState 객체에 자원/플래그/현재 노드 저장
- 선택지 효과: 선택 시 effects 객체의 값을 gameState에 적용
- 세이브/로드: localStorage에 gameState JSON 저장, 자동 저장(매 선택 후)
- 엔딩 도감: localStorage에 도달한 엔딩 목록 저장, 별도 화면에서 조회
- 반응형: 세로 레이아웃, 텍스트 크기 rem 기준 조절
- 접근성: 키보드로 선택지 이동 가능 (위아래 방향키 + 엔터)
