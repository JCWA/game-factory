# 004 - 플래피 버드

## 컨셉
파이프 사이를 통과하는 새. 원터치 조작의 중독성 있는 게임.

## 게임 규칙
- 새가 중력으로 떨어짐
- 클릭/탭하면 위로 점프
- 파이프 사이 틈을 통과하면 1점
- 파이프나 바닥/천장에 닿으면 게임오버
- 파이프 간격 랜덤 변화

## 조작법
- **PC**: 스페이스바 또는 클릭
- **모바일**: 탭

## 비주얼
- 밝은 하늘 배경 그라데이션
- 귀여운 새 (Canvas로 그린 원형)
- 초록색 파이프
- 흐르는 구름 배경
- 바닥 스크롤

## 기술
- Canvas API
- 중력 + 점프 물리
- 단일 index.html
