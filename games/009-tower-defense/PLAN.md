# 009 - 타워 디펜스

## 컨셉
경로를 따라 진행하는 적을 타워로 막는 전략 게임. 웨이브 기반, 타워 업그레이드.

## 게임 규칙
- 적이 정해진 경로를 따라 기지로 이동
- 경로 옆에 타워를 배치하여 적 처치
- 적 처치 시 골드 획득 → 타워 구매/업그레이드
- 기지 HP가 0이 되면 게임오버
- 웨이브가 진행될수록 적이 강해짐

## 타워 종류 (4종)
| 타워 | 비용 | 특성 |
|------|------|------|
| 아처 | 50 | 빠른 공격속도, 단일 타겟 |
| 캐논 | 100 | 느리지만 범위 데미지 |
| 프리즈 | 75 | 적 이동속도 감소 |
| 스나이퍼 | 150 | 긴 사거리, 높은 데미지 |

## 적 종류
| 적 | HP | 속도 | 특성 |
|----|----|----|------|
| 일반 | 낮음 | 보통 | 기본 |
| 빠른 | 낮음 | 빠름 | 회피 |
| 탱크 | 높음 | 느림 | 방어력 |
| 보스 | 매우 높음 | 느림 | 5웨이브마다 등장 |

## UI 구성
```
┌──────────────────────────┐
│ Wave: 1  Gold: 200  ❤ 20 │
├──────────────────────────┤
│                          │

│ [아처] [캐논] [프리즈] [스나이퍼] │

```

## 맵
- 15x12 그리드
- S자형 경로
- 경로 외 영역에 타워 배치 가능

## 기술
- Canvas API
- 적 경로 추적 (웨이포인트)
- 타워 타겟팅 AI (가장 앞선 적 우선)
- 투사체 시스템
