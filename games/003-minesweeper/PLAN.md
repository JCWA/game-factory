# 003 - 지뢰찾기

## 컨셉
클래식 지뢰찾기의 깔끔한 현대 버전. 3단계 난이도.

## 게임 규칙
- 격자에 숨겨진 지뢰를 피하며 안전한 칸을 모두 열기
- 숫자는 주변 8칸의 지뢰 수를 표시
- 빈 칸(주변 지뢰 0개) 클릭 시 연쇄 오픈
- 깃발로 지뢰 위치 표시 가능
- 지뢰 클릭 시 게임오버
- 지뢰 외 모든 칸 오픈 시 승리

## 난이도
| 난이도 | 격자 | 지뢰 수 |
|--------|------|---------|
| 초급   | 9x9  | 10      |
| 중급   | 16x16| 40      |
| 고급   | 16x30| 99      |

## 조작법
- **PC**: 좌클릭(열기), 우클릭(깃발)
- **모바일**: 탭(열기), 길게 누르기(깃발)

## UI 구성
```
┌──────────────────────────┐
│  💣 10    😀    ⏱ 000    │
├──────────────────────────┤
│                          │
│        격자 보드          │
│                          │
├──────────────────────────┤
│  [초급] [중급] [고급]     │
└──────────────────────────┘
```

## 비주얼 스타일
- 밝은 배경, 플랫 디자인
- 닫힌 칸: 살짝 볼록한 3D 느낌
- 열린 칸: 평평한 밝은 색
- 숫자별 색상 (1=파랑, 2=초록, 3=빨강, ...)
- 게임오버 시 모든 지뢰 표시 + 빨간 하이라이트

## 기술 구현
- CSS Grid로 보드 렌더링
- 첫 클릭 시 지뢰 배치 (첫 클릭 안전 보장)
- 타이머 기능
- 단일 index.html
