# 020 - 중력 샌드박스

## 컨셉
우주 공간에 행성과 별을 배치하여 중력 상호작용을 시뮬레이션하는 실험 게임. 플레이어가 천체를 놓으면 만유인력에 의해 서로 끌어당기며 궤도를 그린다. 안정적인 태양계를 만들거나, 혼돈적인 충돌 쇼를 연출하거나, 주어진 챌린지(위성 궤도 안착, 행성 슬링샷 등)를 풀 수 있다.

## 게임 규칙
- 빈 우주 공간에 천체를 드래그하여 배치한다
- 천체 배치 시 드래그 방향과 길이로 초기 속도 벡터를 설정한다
- 천체 종류: 항성(큰 질량, 발광), 행성(중간), 소행성(작음), 블랙홀(초대질량, 흡수)
- 시뮬레이션 시작 후 뉴턴 만유인력 법칙에 따라 모든 천체가 상호작용
- 천체끼리 충돌하면 합쳐져서 질량이 합산된다 (비탄성 충돌)
- 블랙홀은 일정 범위 내의 모든 천체를 흡수한다
- 궤도 예측선: 배치 시 점선으로 예상 궤적을 미리 보여준다
- 시간 배속: 0.5x, 1x, 2x, 5x, 10x
- 챌린지 모드: 5개의 미션 (안정 궤도 만들기, 목표 지점 도달 등)
- 샌드박스 모드: 자유롭게 실험

## 조작법
- PC: 마우스 클릭+드래그로 천체 배치 (드래그 방향=초기 속도), 스크롤로 줌, 우클릭 드래그로 카메라 이동, 키보드 1-4로 천체 종류 선택
- 모바일: 터치 드래그로 배치, 핀치 줌, 두 손가락 드래그로 카메라, 하단 버튼으로 천체 선택

## 비주얼 스타일
- 배경: 깊은 우주색 (#0a0a1a), 랜덤 별 입자 반짝임
- 항성: 노란~주황 원형 + 외곽 발광 그림자(glow), 크기는 질량에 비례
- 행성: 파랑/초록/빨강 등 다양한 색상, 항성보다 작음
- 소행성: 회색 작은 점
- 블랙홀: 검정 원 + 보라색 이벤트 호라이즌 링
- 궤적: 천체 색상과 같은 색의 반투명 점선, 시간이 지나면 서서히 사라짐
- 충돌 시 파티클 폭발 이펙트 (원형으로 퍼지는 입자들)
- 배치 드래그 시 속도 벡터를 화살표로 표시

## 핵심 시스템

### 만유인력 시뮬레이션
```javascript
// 모든 천체 쌍에 대해 중력 계산
// F = G * m1 * m2 / r^2
// 가속도 a = F / m

const G = 0.5; // 조정된 중력 상수

function updatePhysics(bodies, dt) {
  for (let i = 0; i < bodies.length; i++) {
    let ax = 0, ay = 0;
    for (let j = 0; j < bodies.length; j++) {
      if (i === j) continue;
      const dx = bodies[j].x - bodies[i].x;
      const dy = bodies[j].y - bodies[i].y;
      const distSq = dx * dx + dy * dy;
      const dist = Math.sqrt(distSq);
      
      // 최소 거리 제한 (수치 폭발 방지)
      const safeDist = Math.max(dist, bodies[i].radius + bodies[j].radius);
      const force = G * bodies[j].mass / (safeDist * safeDist);
      
      ax += force * dx / dist;
      ay += force * dy / dist;
    }
    bodies[i].vx += ax * dt;
    bodies[i].vy += ay * dt;
  }
  
  // 위치 업데이트
  for (const body of bodies) {
    body.x += body.vx * dt;
    body.y += body.vy * dt;
  }
}
```

### 충돌 감지 및 합체
```
- 매 프레임 모든 천체 쌍의 거리 확인
- 두 천체의 거리 < 반지름 합이면 충돌
- 충돌 시:
  - 새 질량 = m1 + m2
  - 새 속도 = (m1*v1 + m2*v2) / (m1 + m2) (운동량 보존)
  - 새 반지름 = sqrt(r1^2 + r2^2) (면적 보존)
  - 큰 쪽의 타입 유지, 위치는 질량 중심
  - 충돌 파티클 생성
```

### 궤도 예측
```
- 배치 드래그 중 가상 시뮬레이션 실행
- 현재 모든 천체 + 배치할 천체를 복사하여 100스텝 시뮬레이션
- 결과 좌표를 점선으로 표시
- 성능을 위해 예측은 10프레임마다 1회 업데이트
```

## 기술 구현
- Canvas 2D로 전체 렌더링, requestAnimationFrame 루프
- 카메라 시스템: 줌(scale)과 오프셋(translate)으로 뷰 변환
- 배경 별: 캔버스 크기의 200개 랜덤 점, parallax 효과 (카메라 이동 시 느리게 이동)
- 궤적 히스토리: 각 천체의 최근 200개 좌표를 ring buffer로 저장
- 발광 효과: ctx.shadowBlur + ctx.shadowColor로 구현
- 파티클 시스템: 충돌 시 30개 파티클 생성, 매 프레임 속도 감소 + 투명도 감소
- 시간 배속: dt에 배속 값을 곱하여 적용, 높은 배속에서는 서브스텝 사용
- N-body 성능: O(n^2)이지만 천체 50개 이하에서는 충분히 빠름
- 챌린지 모드: 목표 조건(특정 영역에 천체 도달, 안정 궤도 N초 유지 등) 체크
