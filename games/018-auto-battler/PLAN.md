# 018 - 오토 배틀러

## 컨셉
유닛을 구매하고 배치하면 전투는 자동으로 진행되는 전략 게임. 매 라운드 상점에서 유닛을 고르고, 3x4 그리드에 배치한 뒤 "전투 시작"을 누르면 적 진영과 자동으로 싸운다. 같은 유닛 3개를 모으면 상위 등급으로 합성된다. 체력이 0이 되기 전에 최대한 많은 라운드를 생존하는 것이 목표.

## 게임 규칙
- 매 라운드 시작 시 5개의 랜덤 유닛이 상점에 표시된다
- 골드로 유닛을 구매하여 대기석에 놓고, 그리드에 배치한다
- 같은 유닛 3개를 보유하면 자동으로 ★2로 합성 (스탯 2배), ★2 셋이면 ★3 (스탯 4배)
- 유닛 종류 (8종):
  - 전사(근접/탱커), 궁수(원거리), 마법사(광역), 힐러(아군 회복)
  - 암살자(후방 점프), 기사(아군 보호), 레인저(빠른 공격), 넥로맨서(죽은 적 소환)
- 시너지 시스템: 같은 타입 2/4개 배치 시 보너스
  - 전열(전사+기사 2): 방어력 +30%
  - 마법진(마법사+넥로 2): 마법 피해 +40%
  - 속사(궁수+레인저 2): 공격속도 +25%
- 전투 시 유닛은 가장 가까운 적을 자동 공격한다
- 라운드 패배 시 남은 적 유닛 수만큼 체력 감소
- 매 라운드 골드 수입: 기본 5 + 이자(보유골드/10, 최대 5) + 연승/연패 보너스
- 총 플레이어 HP: 100, 0이 되면 게임 오버

## 조작법
- PC: 마우스 드래그로 유닛 배치/이동, 클릭으로 구매/판매, 스페이스바로 전투 시작
- 모바일: 터치 드래그로 유닛 배치, 탭으로 구매, 하단 버튼으로 전투 시작

## 비주얼 스타일
- 체스판 느낌의 그리드, 밝은 회색/진한 회색 교차
- 유닛은 동그란 아이콘 + 이모지 또는 심볼 문자로 표현
- ★ 등급에 따라 테두리 색상: ★1 회색, ★2 파랑, ★3 금색 + 발광 효과
- 전투 시 유닛이 슬라이딩하며 이동, 공격 시 흔들림 애니메이션
- 피해 숫자가 유닛 위에 떠오르며 사라짐 (빨간색 데미지, 초록색 힐)
- 시너지 활성 시 해당 유닛들에 아우라 이펙트
- 배경: 어두운 보라색 (#1a1a2e), 그리드 라인 밝은 보라 (#4a4a8a)

### 전투 AI
```
1. 암살자: 적 후열(가장 먼 행)의 HP가 가장 낮은 유닛으로 점프
2. 기사(도발): 인접 적이 기사를 우선 공격하도록 유도
3. 힐러: 아군 중 HP 비율이 가장 낮은 유닛에게 회복
4. 넥로맨서: 적 유닛 사망 시 50% 확률로 아군 측에 소환 (HP 30%)
5. 기본: 가장 가까운 적 공격, range 내 적이 없으면 이동
```

### 합성 시스템
```
- 대기석 + 그리드 전체에서 같은 유닛 3개 감지
- 자동 합성: 그리드에 있는 유닛 우선 유지, 나머지 제거
- ★2: HP/ATK 2배, 외형 테두리 변경
- ★3: HP/ATK 4배, 외형 발광 + 크기 1.3배
```

### 적 라운드 스케일링
```
- 라운드 1-5: 유닛 2-3개, ★1만
- 라운드 6-10: 유닛 3-5개, ★2 등장
- 라운드 11-15: 유닛 5-7개, ★2 위주
- 라운드 16+: 유닛 7-8개, ★3 등장, 시너지 활성
```

## 기술 구현
- DOM 기반 렌더링 (그리드 셀은 div, 유닛도 div)
- 드래그 앤 드롭: HTML5 Drag API 또는 포인터 이벤트 직접 구현
- 전투 애니메이션: CSS transform + transition으로 이동, CSS keyframes로 공격 모션
- 전투 로직은 setInterval(200ms)로 턴 기반 시뮬레이션
- 상점 새로고침: UNIT_TEMPLATES에서 가중 랜덤 선택
- 게임 상태 관리: phase 변수로 'shop' / 'battle' / 'result' 구분
- 데미지 텍스트: position absolute + CSS animation (위로 떠오르며 fade out)
