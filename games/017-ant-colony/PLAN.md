# 017 - 개미 왕국

## 컨셉
플레이어가 여왕개미가 되어 개미 군집을 성장시키는 실시간 시뮬레이션 게임. 일개미를 생산하고 역할(채집, 탐험, 전투)을 배정하여 식량을 모으고 영토를 확장한다. 다른 곤충의 침입을 방어하면서 최대한 큰 군집을 만드는 것이 목표.

## 게임 규칙
- 여왕개미가 둥지 중앙에 위치하며, 식량을 소비해 일개미를 생산한다
- 일개미에게 3가지 역할 배정 가능: 채집개미(식량 수집), 정찰개미(맵 탐색), 병정개미(방어)
- 맵은 2D 탑뷰로, 안개로 가려져 있으며 정찰개미가 탐색하면 밝혀진다
- 식량 소스(열매, 죽은 벌레 등)가 맵 곳곳에 랜덤 생성된다
- 채집개미는 페로몬 경로를 따라 식량을 둥지로 운반한다
- 30초마다 적 곤충(거미, 딱정벌레)이 둥지를 향해 침공한다
- 병정개미가 적을 막지 못하면 둥지가 피해를 입고 여왕 체력이 감소한다
- 여왕 체력이 0이 되면 게임 오버
- 개미 수에 따라 식량 소비가 증가하므로 균형 잡힌 자원 관리가 핵심
- 점수는 총 개미 수 + 탐색 영역 + 생존 시간으로 계산

## 조작법
- PC: 마우스 클릭으로 역할 배정, 스크롤로 맵 이동, 키보드 1/2/3으로 역할 선택 후 맵 클릭으로 이동 지점 지정
- 모바일: 터치로 역할 배정, 드래그로 맵 이동, 핀치로 줌, 하단 버튼으로 역할 선택

## 비주얼 스타일
- 색상: 땅색(#8B4513) 배경, 녹색 식물, 검정/갈색 개미
- 개미는 2-3px 크기의 점으로 표현하되, 머리/몸통/배 3개 원으로 구성
- 페로몬 경로는 반투명 파란 점선으로 표시
- 안개 영역은 반투명 검정 오버레이
- 식량 소스는 노란/빨간 원으로 표현
- 적 곤충은 빨간색 계열로 위협적으로 표현
- 개미 생산 시 둥지에서 팝 애니메이션

## 핵심 시스템

### 개미 AI (Boid-like 시스템)
```
각 개미 객체:
{
  x, y,          // 위치
  role,          // 'gather' | 'scout' | 'soldier'
  target,        // 목적지 {x, y}
  carrying,      // 운반 중인 식량량
  hp,            // 체력
  state          // 'idle' | 'moving' | 'returning' | 'fighting'
}

이동 알고리즘:
- 채집개미: 페로몬 농도가 높은 방향으로 이동, 식량 발견 시 운반하며 페로몬 분비
- 정찰개미: 랜덤 워크 + 미탐색 영역 선호
- 병정개미: 둥지 주변 순찰, 적 감지 시 공격
```

### 페로몬 시스템
```
- 2D 그리드(셀 크기 10px)에 페로몬 값 저장
- 채집개미가 식량 운반 시 경로에 페로몬 +1
- 매 프레임 페로몬 값 0.99 곱하여 서서히 증발
- 채집개미는 인접 셀 중 페로몬 값이 가장 높은 방향으로 이동 (확률적)
```

### 전투 시스템
```
- 병정개미 공격력: 2, HP: 10
- 거미: 공격력 5, HP: 30 (병정 3-4마리 필요)
- 딱정벌레: 공격력 3, HP: 15 (병정 2마리 필요)
- 전투는 범위 내 자동 공격, 1초 간격
```

## 기술 구현
- Canvas 2D로 맵 렌더링 (requestAnimationFrame)
- 개미 최대 200마리까지 성능 고려, 화면 밖 개미는 간소화 업데이트
- 페로몬 그리드는 별도 오프스크린 캔버스에 그려서 메인 캔버스에 합성
- 안개(Fog of War)는 별도 캔버스 레이어로 관리
- 카메라 시스템: 맵 크기 2000x2000, 뷰포트는 화면 크기
- 터치/마우스 드래그로 카메라 이동, 핀치/스크롤로 줌
- UI 오버레이는 HTML/CSS로 캔버스 위에 배치
- 적 스폰 웨이브는 시간 기반, 난이도 점진 상승
