# 030 - 실시간 전략 (RTS) 미니

## 컨셉
실시간 전략 게임의 정수를 단일 HTML에 압축. 자원 채취, 건물 건설, 유닛 생산, 적 AI 기지 파괴까지 모든 RTS 핵심 요소를 포함. StarCraft/Age of Empires의 핵심 루프를 30분 분량으로 축약. 미니맵, 유닛 선택/그룹, 건물 건설 큐, 적 AI의 웨이브 공격까지 갖춘 본격 RTS.

핵심 재미: 멀티태스킹의 긴장감. 자원 관리, 건물 건설, 군대 운용을 동시에 해야 하는 실시간 의사결정. 적 AI가 점점 강해지는 압박감.

## 핵심 시스템

### 시스템 1: 맵 & 자원
- **맵 크기**: 60x40 타일 (2400 타일). Canvas 렌더링.
- **타일 크기**: 16px
- **카메라**: 뷰포트는 맵의 일부만 표시. 마우스 가장자리/방향키로 스크롤.

- **지형 (절차적 생성)**:
  | 지형 | 비율 | 색상 | 이동 가능 | 건설 가능 |
  |------|------|------|----------|----------|
  | 잔디 | 50% | #6b8e23 | O | O |
  | 흙 | 20% | #8b7355 | O | O |
  | 숲 | 15% | #2e8b57 | O (느림) | X |
  | 물 | 10% | #4682b4 | X | X |
  | 바위 | 5% | #808080 | X | X |

- **자원 노드** (맵에 랜덤 배치):
  | 자원 | 개수 | 매장량 | 채취 속도 |
  |------|------|--------|----------|
  | 광물 | 8~12 | 500 | 5/trip |
  | 가스 | 4~6 | 300 | 3/trip |

- **절차적 맵 생성**:
  1. Perlin noise로 기본 지형
  2. 플레이어 기지 (좌하단) / AI 기지 (우상단) 주변 평지 보장
  3. 각 기지 근처 광물 2개 + 가스 1개 보장
  4. 중앙에 추가 자원 배치

### 시스템 2: 건물 시스템
- **건물 목록**:

| 건물 | 크기 | 비용 (광물/가스) | 건설 시간 | 기능 |
|------|------|----------------|----------|------|
| 사령부 | 3x3 | 시작건물 | - | 일꾼 생산, 자원 반납 지점 |
| 막사 | 2x2 | 150/0 | 8s | 보병 생산 |
| 차고 | 3x2 | 200/50 | 12s | 차량 생산 |
| 연구소 | 2x2 | 200/100 | 15s | 업그레이드 연구 |
| 방어 타워 | 1x1 | 100/0 | 5s | 자동 공격 (사거리 5, 공격 8, 공속 1.5s) |
| 보급고 | 2x2 | 100/0 | 6s | 인구 수용량 +10 |
| 정제소 | 2x2 | 150/0 | 10s | 가스 채취 활성화 (가스 위에 건설) |

- **건설 프로세스**: 
  1. 건물 선택 → 배치 위치 지정 (유효한 위치만 초록 표시)
  2. 가장 가까운 일꾼이 이동하여 건설 시작
  3. 건설 중 HP 게이지 표시 (0% → 100%)
  4. 건설 완료 시 기능 활성화
  5. 건설 중 공격받으면 파괴 가능

### 시스템 3: 유닛 시스템

| 유닛 | 비용 (광/가) | 생산시간 | HP | 공격 | 사거리 | 공속 | 이동 | 인구 |
|------|-------------|---------|----|----|--------|-----|------|------|
| 일꾼 | 50/0 | 4s | 30 | 3 | 1 | 2s | 2.5 | 1 |
| 보병 | 50/0 | 5s | 50 | 8 | 1 | 1.5s | 2.0 | 1 |
| 의무병 | 75/25 | 6s | 40 | 0 | 0 | - | 2.0 | 1 |
| 저격수 | 100/25 | 7s | 35 | 15 | 6 | 2.5s | 1.5 | 2 |
| 경전차 | 150/50 | 10s | 120 | 12 | 4 | 2.0s | 3.0 | 2 |
| 중전차 | 250/100 | 15s | 200 | 20 | 3 | 3.0s | 1.5 | 3 |

- **유닛 조작**:
  - 좌클릭: 유닛 선택
  - 드래그 박스: 범위 내 유닛 전체 선택
  - 우클릭 (빈 땅): 이동 명령
  - 우클릭 (적): 공격 명령
  - 우클릭 (자원): 채취 명령 (일꾼)
  - 우클릭 (아군 유닛): 의무병 → 치료 대상 지정
  - Ctrl+숫자: 그룹 지정 (최대 10그룹)
  - 숫자: 그룹 선택
  - A: 공격 이동 (이동 중 적 발견 시 공격)
  - S: 정지
  - H: 홀드 (이동 안 하고 사거리 내 적만 공격)

- **유닛 행동 (상태 머신)**:
  ```
  Idle → 명령 대기
  Moving → A* 경로를 따라 이동. 매 프레임 다음 타일로 lerp.
  Attacking → 사거리 내 적을 공격. 사거리 밖이면 추격.
  Gathering → 자원 노드까지 이동 → 채취 → 사령부로 귀환 → 반복.
  Building → 건설 위치로 이동 → 건설 진행.
  Healing → 의무병 전용. 가장 가까운 아군 부상 유닛에게 이동 → 치료 (+5 HP/s).
  Fleeing → HP < 20% 시 자동 후퇴 (사령부 방향).
  ```

- **전투 공식**:
  ```
  피해 = 공격력 * random(0.8, 1.2) - 방어력 * 0.3
  최소 피해 = 1
  치명타 확률: 5% → 피해 * 2
  공격 쿨다운 = 공속 (초)
  사거리 검사: 유클리드 거리 ≤ 사거리 * 타일크기
  ```

### 시스템 5: 적 AI

#### AI 전략 모드 (상태 머신)
```
[개발 모드] (0~3분)
  - 일꾼 3명으로 자원 채취
  - 막사 1개 건설
  - 보병 5명 생산

[확장 모드] (3~6분)
  - 보급고 추가
  - 차고 건설
  - 경전차 2~3대 생산
  - 소규모 정찰 (보병 2명을 맵 중앙으로)

[공격 모드] (6분~)
  - 주기적 공격 웨이브 (2분마다)
  - 웨이브 구성: 시간에 따라 강해짐
    - 웨이브 1: 보병 5
    - 웨이브 2: 보병 5 + 경전차 2
    - 웨이브 3: 보병 8 + 경전차 3 + 저격수 2
    - 웨이브 4+: 중전차 포함, 계속 증가
  - 공격 대상: 가장 가까운 플레이어 건물

[방어 모드] (기지 근처 적 발견 시)
  - 모든 군사 유닛 기지로 복귀
  - 방어 타워 추가 건설
  - 위협 해소 후 이전 모드로 복귀
```

#### AI 자원 관리
```
매 초:
  if 일꾼 수 < 원하는 일꾼 수 (시간에 따라 3→8):
    일꾼 생산
  if 인구 < 인구상한 - 3:
    보급고 건설
  if 광물 > 300 && 공격유닛 부족:
    군사유닛 생산 (보병 > 경전차 > 저격수 우선순위)
```

#### AI 경로탐색 (A*)
- 타일 기반 A* (맨해튼 거리 휴리스틱)
- 이동 불가 타일: 물, 바위, 건물
- 숲: 이동 비용 2 (일반 1)
- 경로 캐싱: 같은 출발/도착이면 10초간 캐시 사용

## 게임 규칙
- 시작: 사령부 1개 + 일꾼 4명 + 광물 100/가스 0
- 인구 상한: 사령부 5 + 보급고당 10. 최대 50.
- 건물은 사령부 근처 또는 기존 건물 근처에만 건설 가능 (반경 10)
- 자원은 일꾼이 채취하여 사령부로 운반
- 가스 채취는 가스 노드 위에 정제소 건설 필요
- 건물/유닛 HP 0 → 파괴/사망 (파괴 애니메이션)
- 적 사령부 파괴 시 승리
- 플레이어 사령부 파괴 시 패배
- 안개 시스템: 유닛/건물 시야 범위 (5타일) 밖은 안개
- 유닛은 1프레임에 1타일 이하 이동 (이동속도 = 타일/초)
- 생산 큐: 건물당 최대 5개 유닛 큐
- 동시 선택 유닛 최대 20개
- 자원 부족 시 생산 큐 자동 일시정지
- 전투 중 유닛은 가장 가까운 적을 우선 공격 (수동 타겟 지정 가능)

## 비주얼 스타일
- **색상 팔레트**:
  - 잔디: #6b8e23 ~ #556b2f (변형)
  - 흙: #8b7355
  - 물: #4682b4 (반투명)
  - 숲: #2e8b57 (나무 아이콘)
  - 플레이어 색: #4169e1 (파랑)
  - 적 색: #dc143c (빨강)
  - UI: #1a1a2e 배경, #e0e0e0 텍스트
  - HP바: 초록→노랑→빨강 (비율에 따라)

- **유닛 표현**: 색상 원 (크기는 유닛 타입별). 유닛 타입 아이콘 (텍스트 기반).
  - 일꾼: 작은 원 + ⚒
  - 보병: 중간 원 + ⚔
  - 의무병: 중간 원 + ✚ (흰색)
  - 저격수: 중간 원 + ◎
  - 경전차: 큰 원 + □
  - 중전차: 큰 원 + ■

- **건물 표현**: 사각형 + 테두리. 건설 중 = 반투명 + 진행률 바.

- **선택 표시**: 선택된 유닛 아래 초록 원. 선택 박스 = 초록 점선 사각형.

- **애니메이션**:
  - 유닛 이동: 부드러운 lerp
  - 공격: 유닛에서 타겟으로 작은 발사체 (또는 라인 플래시)
  - 폭발: 건물/유닛 파괴 시 주황 파티클
  - 채취: 일꾼 → 자원 왕복 시 점선 표시
  - 치료: 의무병 → 대상 초록 파티클

- **안개**: hidden = 검정, seen = 반투명 검정 (지형만 보임), visible = 투명

## 레벨/콘텐츠 설계
- 단일 게임 모드 (vs AI)
- 난이도 3단계:
  | 난이도 | AI 자원 보너스 | 웨이브 시작 | 웨이브 강도 |
  |--------|-------------|------------|------------|
  | 쉬움 | x1.0 | 8분 | 약함 |
  | 보통 | x1.2 | 6분 | 보통 |
  | 어려움 | x1.5 | 4분 | 강함 |

- **튜토리얼**: 첫 게임 시 순차 안내
  1. "일꾼을 선택하고 광물을 우클릭하세요" → 채취
  2. "막사를 건설하세요" → 건물
  3. "보병을 생산하세요" → 유닛 생산
  4. "적 기지를 찾아 파괴하세요" → 전투

## 기술 구현 계획

### 도전 과제 및 해결
1. **실시간 루프**: requestAnimationFrame + delta time. 게임 로직은 60fps로 고정. 렌더링은 가변.
2. **A* 성능**: 동시에 20+ 유닛이 경로를 찾을 수 있음. 해결: 경로 요청을 큐에 넣고 프레임당 3개씩 처리. 경로 캐싱 10초.
3. **유닛 충돌 회피**: 간단한 steering. 같은 타일에 유닛이 있으면 인접 빈 타일로 밀어냄.
4. **선택 박스**: mousedown 좌표 기록 → mousemove로 박스 그리기 → mouseup 시 박스 내 유닛 선택.
5. **미니맵**: 별도 작은 Canvas. 유닛은 점, 건물은 작은 사각형. 클릭 시 카메라 이동.
6. **안개**: 별도 Canvas 레이어. 유닛/건물 위치에서 BFS로 시야 타일 계산. 매 이동 시 업데이트.
